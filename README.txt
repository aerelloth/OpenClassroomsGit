Bonsoir ! Ceci est un repository créé pour l'activité du cours sur Git/GitHub.
Il comprend un mini-projet "Kaléidoscope" qui a pour but d'informer tout le monde de la date de mon anniversaire tout en leur flashant la rétine.

Si vous voulez vous amuser avec :
* Pour changer la durée du changement de couleur : modifier "1000" à la ligne 54.
* Pour mettre votre propre date d'anniversaire : modifier la ligne 38.
